Mis Proyectos

ChessApp – API REST

Es un proyecto personal desarrollado desde cero en ASP.NET Core que implementa una API REST siguiendo una arquitectura MVC con una separación clara entre controladores servicios y lógica de negocio permitiendo la gestión de usuarios mediante registro y login y utilizando Supabase con PostgreSQL para la persistencia de datos garantizando una estructura escalable y mantenible.

Ir a la web
Nducopy

Es un proyecto que analiza una carpeta del sistema para detectar imágenes duplicadas comparando su contenido y elimina las repeticiones devolviendo un conjunto de imágenes únicas optimizando el almacenamiento y manteniendo la carpeta libre de duplicados.

Página web – Iglesia Palabra de Vida

Es proyecto de desarrollo y mantenimiento de una página web realizada con WordPress incluyendo la edición de contenido imágenes y elementos visuales así como el despliegue y gestión del sitio en Hostinger.

Ir a la web
Lobos y Ovejas – Simulación de Ecosistema

Es un proyecto desarrollado en C# que simula un ecosistema dinámico basado en turnos donde lobos y ovejas interactúan dentro de un mapa generado aleatoriamente. Implementa programación orientada a objetos, gestión de estados (hambre, hidratación y resistencia) y un sistema de toma de decisiones basado en votaciones que permite a los animales actuar de forma autónoma para sobrevivir, buscar recursos, cazar o escapar, generando comportamientos emergentes dentro del entorno.

GestionTareasDocker – Aplicación Full-Stack de Gestión de Tareas

Es una aplicación full-stack desarrollada con React, TypeScript, Spring Boot y PostgreSQL que permite gestionar tareas mediante una arquitectura cliente-servidor desacoplada. El proyecto está completamente contenerizado con Docker y Docker Compose, e incluye funcionalidades para crear, visualizar, completar y eliminar tareas, además de una interfaz interactiva con modales para mejorar la experiencia de usuario.

KoadGames – Plataforma Web de Venta de Videojuegos

Es una aplicación web desarrollada con ASP.NET Core que simula una plataforma de venta de videojuegos. El proyecto está estructurado en una arquitectura cliente-servidor, separando el frontend desarrollado con HTML, CSS y JavaScript del backend basado en una API REST. Incluye controladores, servicios, repositorios y modelos para gestionar la lógica de negocio, ofreciendo una estructura organizada, escalable y fácil de mantener.
