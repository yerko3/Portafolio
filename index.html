ChessApp – API REST

Es un proyecto personal desarrollado desde cero en ASP.NET Core que implementa una API REST siguiendo una arquitectura MVC con una separación clara entre controladores servicios y lógica de negocio permitiendo la gestión de usuarios mediante registro y login y utilizando Supabase con PostgreSQL para la persistencia de datos garantizando una estructura escalable y mantenible.
